# MVP 收费点构思 - 2026.2.22（修订版）

## 1. 目标用户画像
**核心假设**：有一类用户，他们经常面临数据跨境合规问题，但预算有限，无法长期雇佣律师或购买昂贵的企业级软件。

**分层用户画像**（MVP阶段优先聚焦第一类）：

1. **小型跨境电商卖家（核心目标群体）**
   - 公司规模：1-50人
   - 年营业额：100万-1000万人民币
   - 典型场景：将欧洲/美国客户的订单数据（含姓名、地址、电话）回传国内处理
   - 合规需求频率：每月1-3次（主要集中在新规出台、平台政策变更、新增海外市场、签署DPA、应对平台审核等节点）
   - 痛点：搞不清 GDPR、CCPA、中国《个人信息保护法》的交叉适用，律师咨询费高（500-2000元/次），自己研究费时且易出错

2. **个人跨境卖家（免费版获客群体）**
   - 年营收100万以下，个人卖家、亚马逊/eBay个体户
   - 同样有合规需求（平台强制要求合规隐私政策）
   - 付费意愿低，但基数极大，可作为免费版用户，后续通过按次付费转化

3. **出海SaaS初创公司 & 独立开发者（待验证，暂不作为MVP核心）**
   - 核心需求：隐私政策撰写、SaaS产品合规架构评估、用户协议审查
   - 需求频率较低，场景与跨境电商差异大，MVP阶段暂缓，后期可扩展

## 2. 用户当前解决方案及成本
| 解决方案 | 耗时 | 单次成本 | 风险 | 说明 |
|----------|------|----------|------|------|
| 咨询专业律师 | 1-3天 | 500-2000元 | 低 | 质量高，但成本高，中小用户负担重 |
| 自己研究法规 | 2-5天 | 0元（时间成本） | 高 | 易误判，且无法实时跟进法规更新 |
| 冒险直接传输 | 0天 | 0元 | 极高 | 可能面临GDPR最高4%全球年营收罚款，国内最高5000万罚款 |
| **我们的工具** | **5分钟** | **[待定]元** | **低（需律师复核）** | **替代90%的基础合规工作：用户先用工具做初步筛查，拿到方案初稿后，再找律师仅做最终复核（成本降至200-300元），总成本降低70%以上** |

**重要声明**：工具输出仅为合规参考，不构成正式法律意见，用户需根据自身情况决定是否采纳，我们不承担相关法律责任。

## 3. 价值量化
假设一家跨境电商年发生合规场景需求**5次**（这是多数中小卖家的真实频率）：
- **律师全案咨询**：5次 × 1000元 = 5000元/年
- **我们的工具 + 律师复核**：工具订阅1188元/年（99元/月） + 律师复核 5次 × 250元 = 1250元，总计约**2438元/年**，节省**50%以上**。

**价值主张**：
- 用律师一次咨询费的价格，获得全年7x24小时的即时合规参考 + 实时法规更新；
- 将原本的“从零到一全案咨询”降为“初步筛查+律师复核”，大幅降低合规成本；
- 替代人工查阅法规的繁琐工作，提升效率，降低误判风险。

**核心差异化价值**：法规实时更新。中小用户无法及时跟进国内外法规变动、监管案例，而工具可以实时迭代规则，这是自主研究和单次律师咨询无法替代的。

## 4. 收费模式（推荐混合模式，但需MVP验证）
### 方案A：混合模式（建议MVP测试）
- **免费版**：新用户终身1次免费完整报告（让用户完整体验核心价值，产生付费意愿）
- **按次包**：19.9元/次（测试价）或 29.9元/次（A/B测试，看哪个转化率高），适合极低频用户
- **订阅版**：99元/月，适合高频用户（含无限次查询、优先响应）
- **企业版**：定制报价（包含API接入、专属支持、定制法规库，不设固定价格，根据实际需求报价）

### 方案B：纯按次付费（备选）
- 单次：19.9元
- 10次套餐：149元（有效期2年）
- 50次套餐：599元（有效期2年）

**优点**：门槛低，吸引低频用户。**缺点**：收入不稳定，需设计转化路径。

### 关键定价说明
- **“单次查询”定义**：1次查询 = 1个独立合规场景 / 1份文档（如隐私政策、DPA）的审查，不限页数、不限字数。确保定义清晰，避免纠纷。
- **套餐有效期**：建议2年，降低用户使用焦虑，提升购买率。
- **企业版**：不设固定价格，避免限制收入空间，根据实际服务内容报价。

## 5. 信任建设与风险规避
- **专业背书**：在官网/文档中标注“规则由10年+跨境合规律师参与打磨”“报告基于最新生效的法律法规”，提升可信度。
- **免责声明**：所有输出报告必须包含：“本内容仅为合规参考，不构成法律意见，使用前请咨询专业律师。我们不承担因使用本工具而产生的任何法律责任。”
- **数据安全**：强调用户上传的数据加密处理、不留存、不滥用，消除数据安全顾虑。可承诺“传输加密，阅后即焚”或“仅用于本次分析，不存储”。

## 6. 下一步验证计划
### 6.1 用户访谈（分层样本）
- **样本量**：5-8人，包括：
  - 3名跨境电商卖家（亚马逊、Shopee等）
  - 1名出海SaaS初创负责人
  - 1名独立开发者
  - 1名跨境合规律师（获取专业视角）
- **访谈问题**（除原有问题外，增加）：
  - 你上次遇到数据合规问题是什么时候？最终花了多少钱、多少时间解决的？
  - 你选择合规工具时，最在意的3个点是什么？（准确性/价格/操作简单/数据安全/专业背书等）
  - 你能接受的单次付费、包月付费的最高价格分别是多少？
  - 你对工具输出的报告信任度有多高？是否愿意基于报告做合规决策，还是必须找律师复核？
  - 你会不会因为数据安全顾虑，不愿意上传自己的合同、用户数据到工具中？
  - 除了订单数据回传，你最高频的合规场景还有哪些？（如隐私政策更新、DPA签署、平台审核等）
  - 如果工具不承担法律责任（有免责声明），你还愿意付费吗？

### 6.2 落地页测试
- 制作一个极简落地页（可用Webflow、Unbounce或简单的HTML），包含：
  - 核心价值主张（一句话说明）
  - 核心功能列表（仅2项：跨境订单数据回传合规判断、隐私政策/DPA文档审查）
  - 定价信息（同时展示19.9元/次和29.9元/次两个版本，随机分流，统计点击“立即试用”后的转化率）
  - 专业背书（律师参与、法规来源）
  - 数据安全承诺
  - “立即试用”按钮（点击后跳转到表单或直接进入产品）
- **关键指标**：用户看到定价后的点击转化率、完成付费的比例。

### 6.3 MVP产品极简功能
- **只做两个核心场景**（根据访谈结果优先选一个）：
  1. 跨境订单数据回传合规判断（如“中国用户实名信息传输到欧盟”）
  2. 隐私政策/DPA文档合规审查（用户上传文档，AI分析是否符合GDPR/CCPA/中国法）
- **功能优先级**：准确性 > 操作极简 > 报告清晰。初期不追求美观，命令行输出或简单网页即可。

## 7. 核心假设与风险
### 核心假设
- 混合模式能同时覆盖低频和高频用户，按次付费是主要收入来源，订阅制用于留存。
- 用户对19.9元/次的低价有正向反应，但不会因低价而质疑专业性（需靠专业背书抵消）。
- 用户感受到完整价值（免费版体验一次）后会愿意为后续服务付费。

### 风险与对策
| 风险 | 对策 |
|------|------|
| 低价导致“不专业”印象 | 强化专业背书（律师参与、法规依据）；在落地页A/B测试不同定价 |
| 用户升级订阅的动力不足 | 设计转化路径：累计消费满3次按次后推送升级订阅；订阅用户享额外权益（如模板库、人工复核折扣） |
| 数据安全顾虑 | 明确安全承诺；提供“阅后即焚”选项；初期可让用户手动输入数据而非上传文件 |
| 报告准确性不足，用户不信任 | 先由律师复核部分结果，优化提示词；在报告中标注“建议咨询律师” |
| 收入预期：MVP阶段可能主要来自按次付费 | 先跑通按次付费的转化，再优化订阅留存，不预设过高订阅收入 |
