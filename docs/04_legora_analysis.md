# Legora 商业模式分析 - 2026.2.22

## 1. 产品定位
Legora是一款AI法律助手，核心提供合同自动审查、法律条款分析、合规风险提示功能，产品核心目标为帮助律师、法务群体提升工作效率，降低法律事务的重复劳动成本。

## 2. 目标客户
产品核心目标客群分为三类，具体如下：

- **律师事务所**：重点覆盖中小型律所，该类群体有法律提效需求，但无力负担昂贵的法律科技产品。
- **企业法务部门**：主要满足企业日常合同初步审查等常态化工作需求。
- **初创公司**：覆盖有基础法律支持需求，但专项法律服务预算有限的创业主体。

## 3. 定价模式
根据Legora官网披露信息，产品采用纯订阅制收费模式，分为三个梯度版本，具体规则如下：

- **基础版**：49美元/月/用户，单月限200份文档处理。
- **专业版**：99美元/月/用户，单月文档处理数量无上限。
- **企业版**：定制化报价，包含API接口接入、专属客户支持等定制化权益。

## 4. 核心功能（与我们的原型对比）
| 功能 | Legora | 我们的原型 |
|------|--------|------------|
| 合同条款分析 | ✅ 支持 | ❌ 未实现 |
| 多国法规支持 | ✅ 部分支持 | ✅ 支持（GDPR/CCPA/中国法） |
| 数据导出 | ✅ PDF/Word格式导出 | ⚠️ 仅支持控制台输出 |

## 5. 可借鉴之处
基于Legora的产品布局，有以下3点可借鉴的成熟经验：

- **产品功能聚焦**：核心主打合同审查核心场景，产品价值直观清晰，用户可快速感知核心作用。
- **定价分层清晰**：覆盖从入门级用户、中高频专业用户到大型企业客户的全层级需求，适配不同客群的付费能力。
- **品牌信任体系完善**：官网展示大量客户案例与用户评价，可有效降低新用户的决策门槛，提升产品可信度。

## 6. 差异化机会
结合Legora的产品布局、能力边界与我们的原型现有优势，可落地以下3个核心差异化方向，避免与Legora正面同质化竞争：

- **聚焦垂直赛道**：主打“跨境数据合规”细分领域。Legora核心能力集中在通用合同全场景审查，未针对跨境数据合规做深度垂直深耕，我们可避开通用合同审查的红海竞争，将产品核心资源集中在跨境数据合规专项场景，打造垂直品类的产品心智，形成错位竞争优势。
- **补充灵活定价模式**：覆盖低频需求用户。Legora仅采用固定周期订阅制收费，无其他付费模式，对低频使用、阶段性有需求的用户门槛较高。我们可在订阅制基础上，新增按次付费模式，适配低频使用用户的需求，填补Legora未覆盖的用户群体，降低用户入门决策门槛。
- **深耕中国企业本土需求**：打造双法规交叉分析核心能力。Legora仅部分支持多国法规，未针对中国出海企业的核心需求做专项适配，我们可基于现有GDPR/CCPA/中国法的支持能力，重点打造GDPR与中国《个人信息保护法》等国内法规的交叉合规分析能力，精准匹配中国企业出海的双重合规需求，形成本土适配的差异化优势。
